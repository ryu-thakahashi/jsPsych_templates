# jsPsych_templates
jsPsych で実験する際のテンプレート

## 使い方

### 1. このリポジトリをクローンする

```bash
git clone https://github.com/ryu-thakahashi/jsPsych_templates.git
```

### 2. ディレクトリに移動する

```bash
cd jsPsych_templates
```

### 3. firebase init でプロジェクトを作成する

firebase-cli をインストールしていない場合は，`docs/how_to_install.md` を参照してください．

```bash
firebase init
```
